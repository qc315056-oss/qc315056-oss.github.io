第8节：内置对象

String对象

字符串数据≠String对象 创建String对象：var txt = new String(“string”)

根据索引返回字符串

方法 描述
charAt(索引) 根据索引位置返回对应的字符，字符串第一个字符的索引为0
concat(1个/多个字符串) 连接字符串
replaceAll(字符串1，字符串2) 替换所有，与replace()一样，但是是替换所有
indexOf(所查找字符) 获取字符在完整字符串中第一次出现的索引，如果找不到则返回-1
lastindexOf(所查找字符) 获取字符在完整字符串中最后一次出现的索引，如果找不到则返回-1
replace(字符串1，字符串2) 使用字符串

字符串截取

方法 描述
slice(起始位置索引，结束位置索引) 截取从起始位置索引到结束位置索引之间的1个字符串，若省略结束位置索引则表示从起始到结尾（若含有结束位置索引，在截取时则不包括结束位置索引）
substring(起始位置索引，结束位置索引) 截取从起始位置索引到结束位置索引之间的1个字符串，若省略结束位置索引则表示从起始到结尾（若含有结束位置索引，在截取时则不包括结束位置索引）
substr(起始位置索引，截取长度) 从起始位置索引截取，截取长度决定截取几个字符，若省略length则表示从起始索引截取到字符串末尾

字符分割

split(分隔符号，分割数量) 把字符分隔为字符串数组

字符串转换

只针对英文字母有效 toLowerCase() 把字符串转换为小写 toUpperCase() 把字符串转换为大写

Date对象

创建：var d = new Date() //结果为系统当前时间

方法 描述
getFullyear(value) 设置年份
setMouth(value) 月份
setDate(value) 月份中的某一天
setHours(value) 小时数
setMinutes(value) 分钟数
setSeconds(value) 秒数
getDate() 从Date对象返回一个月中的某一天（1~31）
getDay() 从Date对象返回一周中的某一天（0~6） 0是星期日
getFullYear() 从Date对象以四位数字返回年份
getHours() 返回Date对象的小时（0~23）
getMinutes() 返回Date对象的分钟（0~59）
getMouth() 从Date对象返回月份（0~11） 0是一月，（对象.getMouth + 1）
getSeconds() 返回Date对象的秒数（0~59）

数组对象

concat(数组n)

连接两个或者更多的数组，并返回结果，这个方法可以连接数组也可以连接非数组

1
2
3

var a = [1,2,3];
a.concat(4,5); //返回结果[1,2,3,4,5]
a.concat([4,5]); //返回结果[1,2,3,4,5]

filter

过滤方法，返回符合条件的元素

1
2
3
4

array.filter{
	function(item,index){
		return 条件
	}
}

join

当数组元素为undefined、null或空数组时，对应的元素会被转换为空字符串 作用：无符号分隔数组元素

1
2
3
4
5

var arr = new Array(3);
arr[0] = "0";
arr[1] = "1";
arr[2] = "2";
document.write(arr.join()); //0,1,2

indexOf

在数组中查找元素，找得到返回索引，找不到返回-1 可返回某个指定的字符串值在字符串中首次出现的位置 注意：查找元素索引时，只有要查找的元素值与数组元素值全等时才算查找成功

lastIndexOf

从字符串末尾开始检索

map

map()方法是将数组中的每个元素调用一个提供的函数，结果作为一个新的数组返回，并没有改变原来的数组

1
2
3
4

array.map(
	function(item-数组元素,index-数组索引){
	}
)

reverse

用于颠倒数组中元素的顺序。该方法会改变原来的数组，而不会创建新的数组

slice

slice(start,end) 可从已有的数组中返回决定的元素。返回一个新的数组，包含从start到end的arrayObject中的元素。截取的结果不包含终止索引的值

fill

fill(填充值,start,end) 用一个固定值填充数组中从起始索引到终止索引的全部元素，不包括终止索引

sort

用于对数组的元素进行排序 补充：sort方法对数字进行排序的方法

toString

把数组转换为逗号分隔的字符串

添加或删除数组元素的方法

方法名 描述
push(元素1,…) 数组末尾添加一个或多个元素，会修改原数组，返回值为数组的新长度
unshift(元素1,…) 数组开头添加一个或多个元素，会修改原数组，返回值为数组的新长度
pop() 删除数组的最后一个元素，会修改原数组，若是空数组则返回undefined，返回值为删除的元素
shift() 删除数组的第一个元素，会修改原数组，若是空数组返回undefined，返回值为删除的元素
splice(开始索引，删除数量，添加元素) 从指定索引的删除或添加数组元素，会修改原数组，返回值是一个由被删除的元素组成的新数组

Math对象

创建：Math属性/方法() 属性：PI 返回π 方法 ceil(x) 向上取整 floor(x) 向下取整 max(x,y,z…) 找最大值 min(x,y,z…) 找最小值 random() 生成0~1之间的随机数(包含0不含1) round(x) 四舍五入为最近的整数 Math.random()*(n-m)+m；生成m到n之间的随机数，不包括n Math.floor(Math.random()*(n-m+1)+m)；生成m到n之间的随机数 Math.floor(Math.random()*(n+1))；生成0到n之间的随机整数 Math.floor(Math.random()*n+1)；生成1到n之间的随机整数

定时器

设置定时器

setTimeout(function(){},毫秒时间) 在指定时间后调用函数，执行一次（倒计时） setInterval(function(){},毫秒时间) 按照周期自动调用函数，执行多次（重复执行）

清除定时器

clearTimeout(定时器名称) 清除setTimout()设置的定时器（仅释放内存作用） clearInterval(定时器名称) 清除setInterval()设置的定时器（停止执行并释放内存）
