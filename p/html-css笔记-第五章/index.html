HTML+CSS笔记 第五章

2026-03-28
第5章：表格和表单

表格

<table>定义一个表格的开始和结束

border-spacing-设置格与格之间的距离 左：margin-right:auto 中：margin:0 auto 右：margin-left:auto

属性 描述
border 设置表格的边框[[第4章：盒子模型#边框属性]]
border-spacing 设置单元格与单元格的距离

<tr>定义一行

<tr>内外边距属性（padding和margin） lineheight-行高 align-水平对齐 valign-垂直对齐 只能嵌套<td>和<th>

<td>必须嵌套在<tr>内

text-align-内容水平对齐 vertical-align-内容垂直对齐 colspan-水平合并 rowspan-垂直合并

<th>定义表头

文本默认加粗居中

表格结构 必须按照这个顺序写

<thead>定义表格头部、<tfoot>定义表格页脚、<tbody>定义表格主体

表格边框

border-collapse： separate-双线边框、分离边框 collapse-单线边框、合并边框

表单

创建表单

<form>创建表单、action属性-服务器地址、method属性、name属性

表单控件

input控件

单标记

1

<input type="控件类型">

type： text-单行文本 file-选择文件只能选择，不能上传 password-密码输入框 submit-提交按钮 radio-单选按钮，结合name reset-重置按钮 checkbox-复选框 image-图像形式提交按钮，结合src button-按钮，value定义按钮名称 search-快速清除 color-设置颜色 number -输入数字

属性 属性值 描述
type
name 由用户自定义 用于区分
value 由用户自定义 设置input默认值
readonly readonly 设置input只读
disabled disabled 第一次加载页面时禁用input
checked checked 定义默认选中的单选按钮或复选框
maxlength 正整数 限制英文及数字个数
autofocus true/false 是否自动获取焦点（焦点仅有一个，鼠标点击哪里哪里就是焦点）
