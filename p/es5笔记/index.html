第3节：运算符与表达式

算术运算符与算术表达式

二元运算符（需要两个运算子）：+、-、*、/、% 一元运算符（需要一个运算子）：++自增、–自减 运算符前置：返回计算后结果、先自增/减再运算 运算符后置：返回计算前结构、先运算后自增/减

字符串运算符

js中，加号“+”可以用于字符串拼接 显示转换与隐式转换的结果如下： 显示转换：也就是强制类型抓换，利用相关转换方法完成 隐式转换：当操作的两个数据类型不同时，js会按照既定规则进行自动转换 转换规则：当操作的两个数据类型不同时，观察数据之间的符号如果为加号“+”，则转换为字符串数据（累加），如果为其他算术运算符，则转换为数字型数据

比较运算符与比较表达式

其结果返回是一个布尔值 以x=5来表示：

比较运算符 描述 比较 返回值
== 等于 x==5 true
=== 绝对等于 x===“5” false
（值和类型） x===5 true
!= 不等于 x!=6 true
!== 不绝对等于 x!==“5” true
（值和类型有一个不等，或者两个都不相等） x!==5 false

==、!=在比较不同类型的数据时，首先会自动将要比较的数据转换成相同的数据类型，然后再进行比较 ===、!==在比较不同类型的数据时，不会进行数据类型的转换

赋值运算符

=：将右边的表达式的值赋给左边的变量，不仅限于数值 +=：将左边的变量加上右边变量再赋给左边

逻辑运算符与逻辑表达式

逻辑运算符共有三个，分别是”!“（逻辑非），”&&（逻辑与）和”||“（逻辑或） &&：只有当两个操作数都为true时，结果才为true，只要有一个为false，结果都为false ||：只要有一个操作数为true，结果就返回true &&：都真才真，有假就假 ||：有真就真，都假才假 !：真变假，假变真

第6节：JS数组

数组是存储多个值的集合，它是由一个或多个元素组成的，各元素直接按使用逗号分隔 数组中的每个元素由索引和值组成 索引：也可称为下标，用于识别元素，以数字表示，默认情况下从0开始依次递减 值：元素的内容，可以是任意类型的数据

数组的创建

  1. 数组字面量，赋值给数组的值被包含在一对中括号里面，每个值用逗号隔开（var只能用来定义变量，不饿能创建数组）
  2. 用array构造函数创建new Array()方法 var a = new Array() //没有参数，创建一个空数组 var b = new Array(5) //声明并初始化一个长度为5的数组 new Array()与[]的区别： 使用[]可以创建含有空存储位置的数组，使用new Array() 不可以创建含有空存储位置的数组

数组的读和写

  • 使用[]运算符来访问数组中的元素
  • 添加/修改数组元素：数组名[索引] = 值（取决于原本是否有值）
  • 通过数组的length方法得到数组长度 数组名.length; 修改的数组长度大于数组原长度，多余的数组元素将会被舍弃
  • 删除数组中某一个索引处的元素，可以使用delete运算符来操作 delete.数组名[索引]，使用delete删除元素时类似给该元素设置为赋值状态，不会修改数组的长度，也就是它的length属性不变

第7节：函数

函数概述

  1. 函数类别有预定义函数和自定义函数
  2. 通过函数可以将计算工资代码即可，无需重复编写
  3. 函数的定义是由function、函数名、参数、函数体四部分组成的 function是定义函数的关键字 函数名不能以数字开头，且不能是js中的关键字 参数是外界传递给函数的值，多个参数之间使用逗号分隔 函数体是专门用于实现特定功能的主题，由一条或多条语句组成 若在调用函数后想要得到处理结果，在函数体中可以使用return关键字返回，只能返回一个结果，且return后不能再由任何语句

参数设置

含有默认值的参数 函数定义：Num(a=3,b)
调用Num(1,2) 1,2
调用Num(1) 1,undefined
调用Num(,2) 写法错误
调用Num(undefined,2) 3,2

函数的调用：在哪里调用就return到哪里

要想在程序中发挥函数的作用，必须得调用这个函数 函数名称(实参名1,实参名2…) 函数声明和函数调用编写的顺序不分前后

变量的作用域

js会根据作用域使用范围的不同，可以将其划分为全局作用域、函数作用域 全局变量：不在任何函数内声明的变量（显式定义）成为全局变量，它在同一个页面文件中的所有脚本内部都可以使用 局部变量：在函数体内利用var关键字定义的变量称为局部变量，它仅在该函数体内有效

匿名函数

普通函数会在代码执行前被加载到作用域中，匿名函数是代码执行到哪一行才会有定义 调用：

  1. 调用时使用变量名()
  2. 使用小括号直接包裹匿名函数，相当于自动调用且只能调用一次(function(形参){执行体}(实参);)
  3. html页面元素单击事件(onclick)

闭包函数

闭包是指有权访问另一个函数作用域中的变量的函数 作用：在函数外部读取函数内部的变量，让变量的值始终保持再内存中，知道页面关闭 注意：因为闭包会使得函数的变量一直保存在内存中，内存消耗很大，所以闭包的滥用会降低程序的处理速度，造成内存消耗等问题

函数的嵌套调用

在内部定义的函数只能在内部调用，，如果在外部调用，就会出错 函数嵌套会形成作用域链（作用域同样嵌套），在作用域链中查找相关变量的值，会按照从里到外的顺序进行知道全局作用域

函数的递归调用：使用场景：阶乘、累加、递归

递归函数用于让一个函数从其内部调用其本身

对象的定义

在js中，对象的定义是通过{}语法实现的。对象的成员以键值对的形式存放在{}中，多个成员之间使用逗号隔开 冒号左边是键名，右边是键值 语法：对象的定义是通过{}语法实现的 组成：对象以对象成员构成，对象成员分为属性和方法，多个成员之间使用逗号分隔 成员：对象的成员以键值对的形式存放在{}中 一对{}表示对象 属性形式：名：值 方法形式：名：匿名函数 创建方法1：{}语法，又称为对象的子民阿亮语法 创建方法2：new Object() 创建方法3：构造函数

访问对象成员

通过”.“访问对象的成员 语法：对象.成员→访问属性 对象.成员（）→访问方法，访问并不存在的成员并赋值那就是在添加对象成员 可变成员名语法：对象[变量名]=值→访问属性 对象[变量名]()→访问方法 原理：用一个函数批量化创建对象 标志：new 其特点在于这些对象都基于同一个模板创建，同时每个对象又有自己的特征

利用构造函数创建对象

第9节：DOM

DOM

文档对象模型 是一个表示和处理文档的应用程序接口（API），可用于动态访问、更新文档的内容、结构和样式

DOM树

代表整个的HTML文档结构 DOM HTML指的是DOM中为操作HTML文档提供的属性和方法 文档表示HTML五年间，文档中的标签称为元素，文档中的所有内容称为节点，一个HTML文件可以看作是所有元素组成的一个节点树，各元素节点之间有级别的划分。元素一定是节点，但节点不一定是元素

DOM树中的界定啊

  1. 根节点：<html>标签是整个文档的根节点，有且仅有一个
  2. 子节点：指某个节点的下级节点
  3. 父节点：指某个节点的上级节点
  4. 兄弟节点：同属于一个父节点的两个子节点

节点类型

元素节点：标签、属性节点：class、id、文本节点：文字

节点方法

创建节点：document.createElement(“标签名”)； 添加节点：appendChild(“元素节点”)表示将一个节点添加到父节点的所有子节点的末尾 insertBefore(“元素节点”,“指定子节点”)表示将节点添加到父节点中的指定子节点的前面，该方法需要接收两个参数：第一个参数表示要添加的节点，第二个参数表示父节点中的指定子节点 移除节点：removeChild(“子节点”)方法将一个父节点的指定子节点溢出 复制节点：cloneNode(true/false)该方法会返回节点对象的副本。该方法的参数是可选参 数：默认为false，表示只复制节点本身，不复制节点内部的子节点。如果设置为true，表示复制节点本身及里面所有的子节点
