笔记 微信小程序

微信小程序

2026-03-29

基本

  1. 开发小程序之前首先删除模板中的冗余代码。 pages文件夹 app.js app.json app.wxss
  2. < view > 模块 相当于div
  3. < text > 文本 相当于p
  4. < input > 输入框 type属性：键盘类型
  5. 定义样式（同CSS一致，但是不能用 * 表示所有） id class 标签
  6. 微信小程序的通用单位：rpx
  7. 公共样式要写在app.wxss里
  8. json文件：对页面效果进行设置

js怎么写

  1. 在组件上绑定事件要用bind
  2. 单击事件tap
  3. 获取value改变事件change
  4. JS中的数据想要显示在wxml中——数据绑定：{{变量名}}，且需要将数据定已在JS文件的data中
  5. 只是在JS文件中作为逻辑处理的数据，不需要定义在data中
  6. 更新data中的数据值要使用setData()，不能直接赋值
  7. JS中处理逻辑时需要使用本页面中的内容，前面要加this.
  8. 方法书写格式： 方法名:function(){}
  9. 在wxss中使用page就相当于操作body，是对页面整体来进行样式处理
  10. 布局处理的相关代码含义 display:flex; 表示使用flex布局 flex-direction:column; 表示沿垂直方向从上到下排列 flex-direction:row; 表示沿水平方向从左到右排列 flex:1; 表示页面中的view平分整个页面的高度 flex-basis:数字%; 表示元素占用的比例 以下3行代码可以使文字水平垂直居中： align-items:center; 表示文字水平居中 justify-content:center; 表示文字垂直居中 display:flex;
  11. 字号的单位：pt
  12. data-自定义属性名=“自定义属性值” 通过event获取对应的值 event.target.dataset.自定义属性名
  13. 更新2个单位： vw：相对屏幕宽度 vh：相对屏幕高度
