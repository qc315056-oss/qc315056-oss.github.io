元素的浮动

元素的浮动属性

元素的浮动是指设置了浮动属性的元素会脱离标准文档流的控制而移动到其父元素指定位置的过程 left-使元素向左浮动、right-使元素向右浮动、none-使元素不浮动，默认值

清除浮动 

1

选择器{clear:属性值}

left-不允许左侧有浮动元素、right-不允许右侧有浮动元素、both-同时清除左右两侧浮动元素的影响

使用空标签清除浮动

在浮动元素后添加空标签，并对空标签应用"clear:both"样式，可清除浮动元素的影响

使用overflow属性清除浮动

为元素应用"overflow:hidden"，谁有影响给谁加

使用after伪对象清除浮动

  1. 为需要清除浮动的元素伪对象设置"height:0"
  2. 为伪对象设置content:"";

overflow属性

处理溢出内容的显示样式

属性值 描述
visible 内容不会被修剪，会呈现在标签框之外，为默认值
hidden 溢出内容修剪，被修剪的内容不可见
auto 需要时显示滚动条
scroll 溢出内容修剪，始终显示滚动条

元素的定位

元素的定位属性 

1

选择器{position:属性值}

static-静态定位，默认、relative-相对定位、absolute-绝对定位，、fixed-固定定位 子绝父相

边偏移

必须结合定位使用 top-定义顶端偏移量、bottom-定义底部偏移量、left-定义左侧偏移量、right-定义右侧偏移量

相对定位

不脱离标准文档流，在不设置偏移量的情况下相当于静态定位

绝对定位

脱离标准文档流 定义多个边偏移属性时，如果left和right的值冲突，以left的值为准，如果top和bottom的值冲突，以top的值为准

固定定位

脱离标准文档流 应用：导航回到页面顶部

z-index层叠等级属性

只能在定位元素上生效，且对父子关系元素生效 默认状态下z-index属性的值为0

元素的类型与转换

元素的类型

  1. 块元素：独占一行的元素 独占一行显示 可以设置宽高和内外边距 可以容纳其他块元素和行内元素 常见的块元素有：h1~h6、p、div、ul、ol、li、table、form、dl等
  2. 行内元素：和内容一样大的元素 宽高由内容决定 单独设置宽高无效 只能设置水平方向的外边距，内边距可以正常设置 只能容纳其他行内元素和文本 常见的行内元素有：strong、b、em、i、del、s、ins、u、a、span、font、label、select、textarea等

<span>元素

  1. 用来单独定义文本
  2. 用来处理除<a>以外的元素 <span>开始标签与</span>结束标签之间只能包含文本和各种行内标签

元素的转换

inline：块→行内 block：行内→块 inline-block：行内，块→行内块

  • 综合块元素与行内元素的特点
  • 宽高由内容决定，可以设置宽高、行高、内外边距
