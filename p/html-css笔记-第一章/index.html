第1章：HTML5标签和属性

标签标签<h1>~<h6> 

1

<hn align ="对齐方式">文本</hn>

align为可选属性，指定对齐方式，属性值：left、center、right 注意： （1）一个页面中最好只使用一个<h1>标签 （2）切勿为了设置文字加粗或更改恩子的大小而使用标题标签

段落标签<p> 

1

<p align ="对齐方式">文本</p>

用<p>来定义段落，段落中的文本会根据浏览器窗口大小自动换行

水平线标签<hr/> 

1

<hr 属性="属性值"/>

hr属于单标记

align 设置对齐方式 left、right、center(默认)
size 设置粗细 以px为单位，默认为2px
color 设置颜色 英文名称、十六进制、rgb
width 设置宽度 像素值、百分比（默认100%）

换行标签

文本格式化标签

均为双标签

<strong></strong> 加粗
<ins></ins> 下划线
<em></em> 斜体
<del></del> 删除线
<cite></cite> 斜体，作品标题专用
<time></tiem> 标注时间
<math></math> 文本添加底色（默认黄色）

文本样式标签<font> 

1

<fonr 属性=“属性值”>文本</font>
face 设置字体
size 设置字号，可以取1~7整数值，无单位
color 设置颜色

特殊字符

空格—  ©—版权字号—© ⁿ—上标—$supn;

图像标签 

1

<img src="图像url">

单标记

alt 设置不能显示时的文本
title 鼠标悬停时显示文本
width 设置宽度
height 设置高度
align=left、top、middle、bottom
top：图像顶端与文本第一行对齐，其他文字位于下方
middle：图像中线与文本第一行对齐，其他文字位于下方
bottom：图像底部与文本第一行对齐，其他文字位于下方

相对路径

  • 位于同一文件夹，只需输入名称
  • 位于下一级，用/
  • 位于上一级，用../

超链接 <a> 

1

  • _self 在原窗口中打开（默认）

结构标签

<header> 存放logo、标题等
<nav> 菜单
<footer> 存放版权声明等

页面交互标签

  1. <details>和<summary>标签 <details>可以让一段内容包含隐藏的信息 <summary>可以嵌入<details>中，显示隐藏信息
  2. <progress>标签 定义进度条，可配合JS实现动画效果 value——已完成的进度、max——全部进度
  3. <meter>标签 value——实际数据 min——数据的最小值、 max——数据的最大值 low——衡量标准的最小值（及格分） high——衡量标准的最大值（优秀分） optimum——大于high，代表越大越好；小于low，代表越小越好
