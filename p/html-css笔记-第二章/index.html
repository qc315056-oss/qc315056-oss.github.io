字体样式

font-size字号

em——倍率单位 px——像素单位 %——百分比单位

font-family字体

可以同时指定多个字体，中间要用”，“隔开 中文字体名需要用英文引号引起 如果名称中包含空格、#等特殊符号，必须用英文引号引起来

font-weight文字粗细

normal——标准、bold——粗体、bolder——比粗体更粗、lighter——比粗体更细

font设置综合字体样式 

1

选择器{font:font-style font-weight font-size / line-height font-family}

必须按以上顺序书写

@font-face规则 

1
2
3
4

@font-face{
	font-family：字体名称;
	src:字体路径;
}

font-style字体风格

normal：默认值 italic：斜体

文本外观属性

letter-spacing 字间距-定义字母

以px、em为单位，0为基点，正数放大，负数缩小

word-spacing 单词间距-定义单词

定义英文单词的间距，对中文字符无效

line-height 行间距

以px、em、%为单位

text-transform 文本转换

none——不转化，默认 capitalize——首字母转换为大写 uppercase——全部字母转换为大写 lowercase——全部字母转换为小写

text-decoration文本装饰

none——没有装饰，默认 underline——设置下划线、line-through——设置删除线 → 可同时设置 overline——设置上划线

text-align 水平对齐方式

left（默认）、right、left

text-indent 首行缩进

通常以em为单位，默认2em 仅适用块元素，对行内元素无效 [[第6章：网页布局#元素的类型与转换]]

white-space 空格处理

normal：默认，文本中的空格换行无效，只显示一个空格，满行自动换行 pre：按照书写格式显示 nowrap：空格和换行无效，除非使用<br>

text-shadow 阴影效果

选择器{text-shadow: h-shadow v-shadow blur color;} h-shadow:水平阴影 v-shadow:垂直阴影距离 blur:模糊半径

text-overflow 处理溢出的文本

!!!不要忘记定义宽度!!! ellipsis——修剪、显示"…" clip——修剪、不显示"…" 格式：(缺一不可)

  1. 为文本元素定义宽度
  2. white-space:nowrap——强制不换行
  3. overflow:hidden——隐藏溢出文本
  4. text-overflow:ellipsis——显示"…"

word-wrap属性

normal：默认值，只在允许的断点字换行 break-word：在长单词或URL内部换行

list-style-type属性

none：不使用列表项目符号 disc：实心圆形（无序列表） square：实心方块（有序列表）

list-style-image属性

为各个列表项设置列表项目图像，通常为<li>标签设置背景图像的方式添加项目图像，属性的值为图像的URL [[第4章：盒子模型#背景属性]]

list-style属性

list-style:列表项目符号 列表项目符号URL 列表项目图像
